# Data

This project uses the **DataCo Smart Supply Chain Dataset** from Kaggle.

- **Source:** (https://www.kaggle.com/datasets/saicharankomati/dataco-supply-chain-dataset)
- **License:** Unknown/unspecified — not redistributed in this repo as a precaution
- **Size:** 95.73MB (raw CSV)

## To reproduce this project locally

1. Download the dataset from the Kaggle link above
2. Place the CSV file in this `data/` folder
3. Update file paths in the SQL scripts / notebooks if needed to match your local setup

All SQL queries and analysis in this repo were run against this raw file using the schema defined in `sql/schema.sql`.
