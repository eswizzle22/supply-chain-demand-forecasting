# Supply Chain Demand Forecasting

The goal of this project is to make the case for an AI-driven demand forecasting model — and show, with real data, why it's worth building. A model like this helps a business order the right quantity of supply to actually match demand, instead of guessing. That means fewer inefficiencies, better accuracy, lower operational costs, and more revenue captured rather than lost.

I'm using a Kaggle supply chain dataset to build that case — first through SQL and dashboards, and eventually with the forecasting model itself.

It's still a work in progress, so consider this a living document.
